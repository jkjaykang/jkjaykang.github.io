This website is something I created to practice Bootstrap and website-making in general. I love writing and drawing about fantasy world building and setting, so I decided to theme it that way. I had a great time making this, so I hope that you can enjoy viewing and reading it too.

Besides this website I have made other various websites to practice different libraries and tools. If you are interested check out those websites which are listed below too.

Have a great day!

Ita's Blacksmith

The idea of this website was to build a shop interface resembling one you would see in a game. To match that idea I used pixel art, which I think came out pretty nice. Pixel art is very cool, and hopefully I can get better at it. On the technical side, the focus of this website was use of JQuery and JQuery-UI and its various tools (draggable, droppable, tooltips, etc.)
